IntelliJ IDEA项目目录的Java类文件名的左下角出现红圈

Posted Mar 18 2021-03-18T05:23:00+00:00 by Lemon
Updated Mar 18 2021-03-18T08:33:48+00:00

报错信息

src目录下的Java类的名字的左下角出现红圈并且中间有一个字母J。

按住CTRL点左键的时候出现“cannot find declaration to go to”。

报错原因

模块路径错误。

解决方法

File –> Project Structure –> Modules进入模块设置界面，重新添加工程路径即可。

参考：idea的src目录下的java类文件名的左下角出现红圈中间有一个小j字母的解决方法（完美解决）
