CentOS安装JDK

Posted Jun 30, 2021
By
1 min read

当前JDK最新版本下载地址：http://www.oracle.com/technetwork/java/javase/downloads/index.html
历史JDK版本下载地址：http://www.oracle.com/technetwork/java/javase/archive-139210.html
Linux上一般会安装Open JDK，关于OpenJDK和JDK的区别：http://www.cnblogs.com/sxdcgaq8080/p/7487369.html

卸载系统自带OpenJDK

查看系统自带的OpenJDK版本信息

1

java -version

查看系统自带Java相关文件

1

rpm -qa | grep java

以上文件中，下面这几个可以删除

1
2
3
4

java-1.7.0-openjdk-1.7.0.111-2.6.7.8.el7.x86_64
java-1.8.0-openjdk-1.8.0.102-4.b14.el7.x86_64
java-1.8.0-openjdk-headless-1.8.0.102-4.b14.el7.x86_64
java-1.7.0-openjdk-headless-1.7.0.111-2.6.7.8.el7.x86_64

删除系统自带Java文件

1
2
3
4

rpm -e --nodeps java-1.7.0-openjdk-1.7.0.111-2.6.7.8.el7.x86_64
rpm -e --nodeps java-1.8.0-openjdk-1.8.0.102-4.b14.el7.x86_64
rpm -e --nodeps java-1.8.0-openjdk-headless-1.8.0.102-4.b14.el7.x86_64
rpm -e --nodeps java-1.7.0-openjdk-headless-1.7.0.111-2.6.7.8.el7.x86_64

检查是否已经删除成功

1

java -version

显示未找到命令...，代表已经删除成功了，如果还没删除，使用yum -y remove xxx命令删除

解压JDK

1

tar -zxvf jdk-8u144-linux-x64.tar.gz

配置环境变量

1

vi /etc/profile

在文本的最后一行插入

1
2
3
4
5

#java environment
export JAVA_HOME=/usr/soft/jdk1.8.0_65
export JRE_HOME=$JAVA_HOME/jre
export PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:$JRE_HOME/lib

退出vi

1
2
3

ESC
shift+q
qw!

刷新环境变量

1

source /etc/profile

检查是否安装成功

1

java -version
